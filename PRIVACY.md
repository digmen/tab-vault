# Privacy Policy — Tab Vault

**Last updated: May 17, 2025**

## Summary

Tab Vault does **not** collect, transmit, or share any personal data. Everything the extension does stays entirely on your device.

---

## Data collected

**None.** Tab Vault does not collect any personal information, browsing history, or usage statistics.

## Data stored locally

Tab Vault saves the following data to your browser's local storage (`chrome.storage.local`) solely to provide its core functionality:

| Data | Purpose |
|---|---|
| Tab URL | To restore the tab later |
| Tab title | To display in the saved list |
| Tab favicon URL | To display the site icon |
| Tab group name & color | To preserve visual grouping |

This data **never leaves your device**. It is not transmitted to any server, third party, or analytics service.

## Permissions used

| Permission | Why it is needed |
|---|---|
| `tabs` | To read tab URL, title and favicon; to open and close tabs |
| `storage` | To persist the saved tab list across browser sessions |
| `contextMenus` | To show "Park this tab" in the right-click menu |
| `tabGroups` | To read tab group name and color when parking a grouped tab |

No host permissions are requested. The extension never reads the content of any web page.

## Remote code

Tab Vault does not load or execute any remote code. All extension code is bundled locally and reviewed by the Chrome Web Store.

## Third parties

Tab Vault does not use any third-party services, analytics tools, advertising networks, or tracking libraries.

## Changes to this policy

If this policy changes, the updated version will be published in this repository with a new "Last updated" date.
